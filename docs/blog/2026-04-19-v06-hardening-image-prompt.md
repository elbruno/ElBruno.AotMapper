# Image Prompt: ElBruno.AotMapper v0.6 — Hardening & AOT

## Hero Image for Blog Post / Social Media

**Theme:** Security, hardening, and speed

**Subject:**
A clean, technical illustration showing a layered lock or shield with a gear or network node inside, suggesting "protected performance." The design should feel modern and minimalist, not cartoonish.

**Visual Style:**
- Clean geometric shapes, sharp lines
- Color palette: Deep blue (#1a3a52), bright teal (#00d4ff), and white on a neutral light gray background
- Tech aesthetic similar to Azure, GitHub, or VS Code branding
- No logos, no text overlays, no photorealism

**Composition:**
- Center: A stylized circuit board or network architecture forming a lock/shield
- Background: Subtle gradient or light pattern suggesting speed/flow (e.g., motion lines, graph grid)
- Aspect ratio: 16:9 (for blog headers / social media)
- 1200px × 675px minimum resolution

**Mood:**
- Professional, trustworthy, forward-looking
- Convey: "hardened," "secured," "optimized"
- Not aggressive, not playful—sophisticated

## Negative Prompts (What to Avoid)

- No text, labels, or code snippets embedded in the image
- No logos (no GitHub logo, no .NET logo, no NuGet logo)
- No cartoonish characters, mascots, or anthropomorphic elements
- No photorealistic images or photographs
- No 3D rendering or heavy textures
- No bright neon colors (avoid reds, cyans that feel "hacker-y")
- No crowded compositions—keep it clean and minimal
